## 🎯 Objetivo do Projeto
Desenvolver uma plataforma digital que atue como uma ponte direta entre a agricultura familiar e o conhecimento técnico-científico. O sistema visa conectar produtores rurais a redes de apoio (universidades, institutos e secretarias), facilitando o acesso a capacitações especializadas e consultorias técnicas para fortalecer a produção sustentável e a comercialização em feiras locais.

---

## 💡 Justificativa
No cenário atual das feiras agroecológicas locais, a comunicação e a transferência de conhecimento técnico para o campo acontecem de forma fragmentada e informal, dependendo quase exclusivamente de ligações e grupos de WhatsApp. 

A ausência de um sistema centralizado resulta em:
* Perda de histórico de atendimentos e cursos;
* Baixa adesão a treinamentos importantes (como controle de pragas e certificações orgânicas);
* Sobrecarga de gerenciamento manual por parte das entidades de extensão universitária e secretarias de agricultura.

A plataforma justifica-se pela necessidade crítica de organizar esse fluxo de dados, garantindo que o conhecimento chegue a quem precisa de forma estruturada, eficiente e centralizada.

## 📂 Estrutura Inicial do Projeto
O projeto encontra-se na sua primeira etapa de desenvolvimento (esqueleto HTML e organização semântica), estruturado da seguinte forma:
* `index.html`: Painel inicial (Dashboard) do produtor/usuário.
* `paginas/`: Arquivos das subpáginas do sistema (`capacitacoes.html`, `consultorias.html`, etc.).
* `css/`: Diretório reservado para as folhas de estilo (`style.css`).
* `js/`: Diretório reservado para os scripts de comportamento (`script.js`).
* `img/`: Diretório reservado para o armazenamento de imagens e logotipos do sistema.
